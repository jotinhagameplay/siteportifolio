Conceitos da Tag HEAD

1. Utilidade da tag HEAD e elementos HTML

A tag HEAD é utilizada para incluir informações sobre o documento que não são visíveis na página. Elementos que podem ser usados dentro do HEAD incluem:

  • <meta>
  • <title>
  • <base>

2. Utilidade e funcionamento das tags meta, title e base

<meta>: Utilizada para fornecer metadados sobre a página. Exemplo:

            
<meta name="description" content="Descrição da página">

<title>: Define o título da página exibido na barra de título do navegador. Exemplo:

            
<title>Título da Página</title>

<base>: Especifica a URL base para todos os links em uma página. Exemplo:

            
<base href="https://www.clonacartao.com/">

3. Charset

Charset (conjunto de caracteres) define o conjunto de caracteres usado na codificação da página. Sua utilidade é garantir que os caracteres sejam exibidos corretamente. Se não definido, o padrão é geralmente UTF-8.

4. Charset Padrão

Se não definido, o charset padrão é geralmente UTF-8.

5. Charsets

UTF-8: Suporta uma ampla variedade de caracteres e é amplamente utilizado.

ASCII: Conjunto básico de caracteres em inglês.

ANSI: Conjunto de caracteres padrão do sistema operacional Windows.

ISO-8859-1: Suporta caracteres usados em muitos idiomas europeus.
